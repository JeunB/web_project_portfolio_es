# 1.Portfolio: Jeunette Berzunza Castillo

Un portafolio web moderno y responsivo que presentará mis habilidades como desarrolladora Full-Stack Junior, construido con tecnologías web fundamentales y técnicas avanzadas de CSS.

# 2.Descripción

Este proyecto es un portafolio personal que muestra mi experiencia en desarrollo web, proyectos destacados y habilidades técnicas. Está diseñado con un enfoque moderno, utilizando efectos visuales avanzados y un diseño completamente responsivo.

# 3.Tecnologías Utilizadas

### Frontend

- HTML5 - Estructura semántica y accesible
- CSS3 - Estilos avanzados y animaciones

### Herramientas de Desarrollo

- Git & GitHub - Control de versiones y despliegue
- Figma - Diseño y prototipado
- VS Code - Editor de código

### Fuentes Personalizadas

- Open Sans - Tipografía principal (Regular, SemiBold, Bold)
- Archivo Black - Títulos y elementos destacados

# 4.Características Destacadas

### Diseño Visual Avanzado

- Efectos de halo luminoso - Gradientes radiales con blur para crear atmósfera
- Diseño responsivo - Adaptación perfecta a desktop, tablet y móvil
- Tipografía personalizada - Fuentes locales optimizadas
- Grid layout moderno - Sistema de tarjetas para proyectos

### Técnicas CSS Avanzadas

- Pseudo-elementos (::before, ::after) - Para efectos visuales complejos
- CSS Grid & Flexbox - Layouts modernos y flexibles
- Media queries - Diseño responsivo en 3 breakpoints
- Gradientes complejos - Efectos de luz y sombra
- Posicionamiento absoluto/relativo - Para efectos de superposición

### Estructura Semántica

- HTML5 semántico - Uso correcto de header, section, article, footer

# 5.Diseño Responsivo

El sitio se adapta perfectamente a diferentes dispositivos:

Desktop (1440px+): Layout completo con efectos visuales
Tablet (768px - 1023px): Adaptación de grid y tipografía
Móvil (hasta 767px): Layout vertical optimizado

# 6.Efectos Visuales

### Halos de Luz

Los efectos de halo utilizan gradientes radiales con múltiples capas:

- Gradientes con rgba() para transparencias
- filter: blur() para efectos de difuminado
- Posicionamiento preciso con calc() y transform

### Sistema de Colores

- Primario: #6d6dff (Azul vibrante)
- Secundario: #4c1a97 (Púrpura profundo)
- Fondo: #000000 (Negro absoluto)
- Texto: #ffffff (Blanco puro)

# 7.Comentarios

Los documentos HTML y CSS se integran con comentarios detallados que pueden llegar a ser muy útiles para un desarrollador junior.
Se usan tanto para distinguir secciones de la página como para detallar la razón del uso de un valor o propiedad específica.

Estos comentarios documentan decisiones de diseño, técnicas CSS avanzadas y la estructura semántica del HTML.
